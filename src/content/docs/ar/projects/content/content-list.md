The content list shows all entries for a collection. Access it from the project sidebar by clicking a collection name.

## List columns

The list displays system columns plus the first few text fields of the collection:

| Column | Description |
|--------|-------------|
| **Status** | `published` or `draft` badge |
| **Locale** | Entry locale (e.g. `EN`, `FR`) |
| **Updated at** | Last modification date |
| Collection fields | Up to 3 text fields from the collection |

## Filtering

Use the filter bar above the list to narrow down entries:

- **Status** — show all, published only, or drafts only
- **Locale** — filter by locale
- **Search** — search by text field values
- **Date range** — filter by creation or update date

## Bulk actions

Select multiple entries using the checkboxes, then choose a bulk action:

- **Publish** — set all selected entries to `published`
- **Unpublish** — set all to `draft`
- **Delete** — move all to trash

## Sorting

Click a column header to sort by that column. Click again to reverse the order. Default sort is by `updated_at` descending.

## Pagination

The list shows 25 entries per page by default. Use the pagination controls at the bottom to navigate.
