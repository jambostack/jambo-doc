استكشف Jambo — نظام إدارة محتوى بلا رأس مفتوح المصدر مبني بـ Symfony 8 و PHP 8.4. يعرض محتويات موقعك عبر واجهة برمجة التطبيقات REST وجهة نهاية GraphQL، مع أدوات ذكاء اصطناعي مدمجة لمساعدتك على تصميم المخططات وإنشاء المحتوى وترجمة الإدخالات تلقائياً.

:::tip[عرض حي متاح]
استكشف Jambo دون تثبيت أي شيء — يعمل عرض مباشر على **[demo.jambostack.site](https://demo.jambostack.site)**.

سجّل الدخول بـ `demo@demo.jambostack.site` / `demo1234` (المصادقة الأساسية أولاً: `demo` / `jambo2026`).
:::

## الميزات الرئيسية

- **واجهات برمجة التطبيقات REST و GraphQL** — يتم إنشاؤها تلقائياً من مجموعاتك
- **AI Studio** — منشئ المخطط القائم على الدردشة
- **وكيل الذكاء الاصطناعي** — العمليات الجماعية
- **دعم متعدد اللغات**
- **خادم MCP**
- **مكتبة الوسائط**
- **الأدوار والأذونات**
- **Webhooks**
- **المصادقة للمستخدم النهائي**
