## ما هو MCP؟

يتيح Model Context Protocol (MCP) لعملاء الذكاء الاصطناعي (Claude، Cursor، العملاء المخصصين) التفاعل مباشرة مع نظام إدارة المحتوى الخاص بك — قراءة المحتوى، إنشاء الإدخالات، إدارة الهيكل — دون كتابة تكاملات مخصصة.

## نقطة النهاية

```
https://نطاقك.com/mcp
```

## فئات الأدوات

| الفئة | الوصف |
|---|---|
| **الاستكشاف** | تصفح المشاريع والمجموعات والهياكل |
| **المحتوى** | سرد، إنشاء، تعديل، حذف الإدخالات |
| **الهيكل** | إدارة المجموعات والحقول |
| **الوسائط** | رفع واستعلام الأصول |
| **المستخدمون النهائيون** | إدارة مستخدمي الواجهة الأمامية |
| **أدوات الذكاء الاصطناعي** | إنشاء المحتوى، البحث، إدارة الإصدارات |

## ربط Claude Desktop

أضف إلى `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "jambo": {
      "url": "https://نطاقك.com/mcp",
      "headers": {
        "Authorization": "Bearer رمز_MCP_الخاص_بك"
      }
    }
  }
}
```

أنشئ رمز MCP في **المشروع → الإعدادات → الوصول إلى MCP**.
