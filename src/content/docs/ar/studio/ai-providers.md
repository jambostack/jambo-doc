## التهيئة

اذهب إلى **الإدارة ← إعدادات التطبيق ← تبويب مزودو الذكاء الاصطناعي** وفعّل المزودين الذين تريد استخدامهم. تُخزَّن مفاتيح الـ API مشفّرةً في قاعدة البيانات — لا حاجة لمتغيّرات البيئة.

كل مزود يتطلب:
- **مفتاح API** (باستثناء Ollama الذي يستخدم رابط URL محليًا)
- **النموذج الافتراضي** — اختر نموذجك المفضل من القائمة المنسدلة

## قدرات المزودين

يكتشف وكيل ذكاء Jambo Studio تلقائيًا ما يمكن لمزوديك المُهيَّأين فعله. يظهر **شارة القدرات** في نافذة دردشة الاستوديو:

```
🟢 نص (claude-sonnet-4-6)  🟢 صور (dall-e-3)
```

| المزود | توليد النص | توليد الصور |
|---|---|---|
| **OpenAI** | ✅ جودة كاملة | ✅ DALL‑E 3 |
| **Anthropic** | ✅ جودة كاملة | ❌ |
| **Google Gemini** | ✅ جودة كاملة | ✅ Imagen |
| **Ollama** | ⚠️ جودة محدودة (تعتمد على النموذج المحلي) | ❌ |
| الباقون | ✅ | ❌ |

إذا كنت بحاجة إلى توليد صور ذكاء اصطناعي لأغلفة المدونة أو صور الأقسام الرئيسية أو صور المؤلفين، فعّل **OpenAI** أو **Gemini** إلى جانب مزود النص الخاص بك. عند عدم توفر أي مزود صور، يستخدم الوكيل صور SVG احترافية كصور بديلة بنظام الألوان الزمردي لـ Jambo.

## Ollama (محلي)

بالنسبة لـ Ollama، أدخل رابط URL للخادم (الافتراضي: `http://localhost:11434`) بدلًا من مفتاح API. هذا يتيح تشغيل ميزات الذكاء الاصطناعي بالكامل دون اتصال بالإنترنت. ملاحظة: تمتلك النماذج المحلية جودة محدودة مقارنةً بمزودي الخدمات السحابية. سيعرض الوكيل تحذيرًا ⚠️ في الدردشة.

## OpenRouter

OpenRouter هو بوابة API موحّدة تمنح الوصول إلى أكثر من 200 نموذج من مزودين متعددين عبر مفتاح API واحد. استخدم تنسيق بادئة النموذج (مثل `openai/gpt-4o`، `anthropic/claude-sonnet-4-6`، `google/gemini-2.0-flash`).

احصل على مفتاح API الخاص بك على [openrouter.ai](https://openrouter.ai/).

## التبديل التلقائي بين المزودين

يجرّب Jambo Studio تلقائيًا المزود الأول المُفعَّل. إذا فشل هذا المزود، يتحوّل إلى المزود التالي — مما يضمن استمرار توفر ميزات الذكاء الاصطناعي.
