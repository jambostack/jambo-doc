يُعدّ وكيل الذكاء الاصطناعي الميزة الأقوى في Jambo Studio. على عكس الدردشة الأساسية (`/schema`، `/data`، `/all`)، يمتلك الوكيل **8 أدوات حقيقية** تُمكّنه من قراءة مخططك وإنشاء المجموعات وتوليد المحتوى بالجملة وترجمة المدخلات وحتى إنتاج صور ذكاء اصطناعي — كل ذلك بأمر واحد.

## وضع الوكيل مقابل الوضع الأساسي

| الميزة | الوضع الأساسي (`/schema`) | وضع الوكيل |
|---|---|---|
| توليد المخطط | ✅ معاينة JSON → تطبيق يدوي | ✅ إنشاء تلقائي + حفظ |
| توليد المحتوى | 3-5 مدخلات لكل مجموعة | **حتى 100 مدخلة دفعة واحدة** |
| متعدد اللغات | يدوي لكل لغة | **تلقائي، جميع اللغات المُهيَّأة دفعة واحدة** |
| الصور | رفع يدوي عبر لوحة الإدارة | **توليد بالذكاء الاصطناعي + رفع تلقائي** |
| العمليات الجماعية | غير متاحة | ✅ إنشاء وتحديث وحذف عشرات المدخلات |
| استكشاف المخطط | سلسلة سياق ثابتة | ✅ أداة `explore_schema()` في الوقت الفعلي |
| الترجمة | غير متاحة | ✅ أداة `translate_entries()` |
| معاينة الخطة | غير متاحة | ✅ PlanCard مع قائمة الإجراءات قبل التنفيذ |

## قدرات المزود

يكتشف الوكيل تلقائيًا ما يمكن لمزود الذكاء الاصطناعي المُهيَّأ لديك فعله ويُكيّف سلوكه. يظهر **شارة القدرات** دائمًا فوق حقل إدخال الدردشة:

```
🟢 نص (claude-sonnet-4-6)  🟢 صور (dall-e-3)
```

عند غياب إحدى القدرات، يُنبّهك الوكيل ويقترح كيفية تفعيلها:

> ⚠️ لم يُهيَّأ أي مزود صور. سأستخدم SVG احترافية كصور بديلة.
> للحصول على صور مُولَّدة بالذكاء الاصطناعي، أضِف OpenAI أو Gemini في **الإدارة ← الإعدادات ← مزودو الذكاء الاصطناعي**.

| المزود | النص | الصور | ملاحظات |
|---|---|---|---|
| **OpenAI** | ✅ GPT-4o, o1, إلخ. | ✅ DALL‑E 3 | قدرات كاملة |
| **Anthropic** | ✅ Claude Opus / Sonnet | ❌ | نص فقط |
| **Google Gemini** | ✅ Gemini Flash / Pro | ✅ Imagen | قدرات كاملة |
| **Ollama** | ⚠️ جودة محدودة | ❌ | نموذج محلي |
| **DeepSeek / Mistral / Groq / xAI / Perplexity / Qwen / OpenRouter** | ✅ | ❌ | نص فقط |

## الأدوات الـ 8 للوكيل

كل أداة وظيفة من جانب الخادم يمكن للذكاء الاصطناعي استدعاؤها. يختار الوكيل الأدوات المناسبة بناءً على طلبك.

### أدوات القراءة (متاحة دائمًا، للقراءة فقط)

| الأداة | الوصف |
|---|---|
| `explore_schema` | تُعيد جميع المجموعات مع حقولها وأنواعها ومؤشرات الحقول الإلزامية والعلاقات. يستدعيها الوكيل في بداية كل جلسة. |
| `read_entries` | تقرأ المدخلات من مجموعة. تدعم الترقيم والتصفية حسب اللغة/الحالة. |

### أدوات الكتابة (قواعد التنفيذ)

| الأداة | التأكيد | ما تفعله |
|---|---|---|
| `create_collections` | **تلقائي** | تُنشئ مجموعات + حقول مباشرة في قاعدة البيانات. تشمل أهداف العلاقات وقيم التعداد. |
| `create_entries` | **تلقائي** | تُنشئ مدخلات بالجملة. تدعم حتى 100 مدخلة × N لغة في استدعاء واحد. |
| `update_entries` | **معاينة مطلوبة** | تُعدّل مدخلات موجودة بالـ UUID. تعرض فرقًا قبل التنفيذ. |
| `delete_entries` | **معاينة + تأكيد** | حذف ناعم للمدخلات بالـ UUID. تعرض قائمة المدخلات المتأثرة قبل التأكيد. |
| `generate_images` | **تلقائي** | تُولّد صورًا عبر DALL‑E أو Gemini وترفعها إلى مكتبة الوسائط وتُعيد الـ UUID. تتراجع إلى SVG إذا لم يتوفر مزود صور. |
| `translate_entries` | **معاينة مطلوبة** | تُترجم المدخلات من لغة مصدر إلى N لغات هدف بالذكاء الاصطناعي. تحافظ على الـ slugs والبيانات المهيكلة. |

### قواعد التأكيد

| الإجراء | السلوك |
|---|---|
| **إنشاء** أي شيء | تنفيذ فوري، لا حاجة لتأكيد |
| **تعديل** بيانات موجودة | عرض معاينة أولًا — انقر "تنفيذ" للتطبيق |
| **حذف** بيانات موجودة | عرض معاينة أولًا — انقر "تأكيد" للحذف |

## كيفية استخدام الوكيل

### 1. الطلبات البسيطة — كشف تلقائي

للطلبات الصغيرة (1-2 مجموعة، 3-5 مدخلات)، استخدم الأوامر القياسية `/schema` أو `/data` أو `/all`. يرد الوكيل في وضع معاينة JSON الكلاسيكي مع أزرار "تطبيق".

```
/schema إنشاء مدونة بمقالات وفئات وتعليقات
/data توليد 5 مقالات مدونة احترافية بالعربية
/all بناء مخطط متجر إلكتروني كامل مع منتجات
```

### 2. الطلبات الجماعية — وضع الوكيل

عند طلب **عمليات جماعية**، يتحوّل الوكيل إلى **وضع التخطيط**:

```
إنشاء موقع وكالة كامل مع محفظة أعمال (6 مشاريع)،
وشهادات (8 اقتباسات)، ومدونة (5 مقالات).
كل شيء بالفرنسية والإنجليزية والإسبانية والعربية.
إضافة صور hero احترافية لكل قسم.
```

يرد الوكيل بـ **PlanCard**:

```
┌────────────────────────────────────────────────┐
│ 📋 الخطة                                       │
│                                                │
│ إنشاء موقع وكالة بمحفظة أعمال،                │
│ شهادات، مدونة — 4 لغات                        │
│                                                │
│ 1. explore_schema                              │
│ 2. create_collections  3 مجموعات               │
│ 3. create_entries     76 مدخلة × 4 لغات        │
│ 4. generate_images     12 صورة                 │
│                                                │
│ [تنفيذ الخطة]                                  │
└────────────────────────────────────────────────┘
```

انقر على **تنفيذ الخطة** وسيُشغّل الوكيل جميع الإجراءات بالتسلسل مع عرض **ExecutionLog**:

```
✅ explore_schema: تم
✅ create_collections: 3 مجموعة
✅ create_entries: 76 مُنشأة، 0 أخطاء
✅ generate_images: 12 صورة مُولَّدة (DALL-E)
```

### 3. الاستكشاف والقراءة

اطلب من الوكيل تحليل مشروعك قبل إجراء التغييرات:

```
استكشف مخططي وأخبرني بما ينقص لمدونة
اقرأ الشهادات الحالية واقترح تحسينات
ما المجموعات التي يجب أن أضيفها لمنصة تعلم إلكتروني؟
```

## نقطة نهاية تنفيذ الأدوات

إذا كنت تبني تكاملًا مخصصًا، يمكنك استدعاء الوكيل برمجيًا:

```
POST /api/projects/{uuid}/studio/ai-execute
Content-Type: application/json
Authorization: Bearer <token>

{
  "actions": [
    { "tool": "explore_schema", "params": {} },
    { "tool": "create_entries", "params": {
        "collection": "testimonials",
        "entries": [{ "quote": "...", "author_name": "..." }],
        "locales": ["fr", "en", "es", "ar"]
    }}
  ],
  "auto_confirm": true
}
```

## نقطة نهاية القدرات

```
GET /api/projects/{uuid}/studio/ai-capabilities
→ { "text": true, "images": true, "provider": "openai", "model": "gpt-4o", "limits": [] }
```

## توليد الصور

يمكن للوكيل توليد الصور عند تهيئة مزود قادر على ذلك (OpenAI أو Gemini).

```
ولّد لقطة شاشة لوحة تحكم داكنة لقسم الـ hero
أنشئ صور غلاف لـ 3 مقالات حول أداء PHP
أضِف صور أفاتار للمؤلفين الستة
```

عند غياب مزود الصور، يتراجع الوكيل إلى **SVG احترافية كصور بديلة** بنظام الألوان الزمردي لـ Jambo.

## نصائح

- **كن محددًا** — «أنشئ 6 شهادات من مطورين مستقلين حول التحوّل من Strapi» أفضل من «أضِف شهادات»
- **اذكر اللغات** — «بالفرنسية والإنجليزية» أو «بجميع اللغات المُهيَّأة» يُشغّل التوليد متعدد اللغات
- **كرّر** — «احتفظ بالشهادات الثلاث الأولى، استبدل الثلاث الأخيرة بأفضل منها»
- **استكشف أولًا** — «حلّل مخططي واقترح تحسينات للمدونة» قبل إجراء التغييرات
- **الصور** — صِف المرئي الذي تريده: «لوحة تحكم داكنة مستقبلية مع لوحات بيانات خضراء ورسوم بيانية تحليلية»
