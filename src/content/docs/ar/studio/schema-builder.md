## نظرة عامة

يتيح لك استوديو الذكاء الاصطناعي في Jambo تصميم مخطط المحتوى وتعديله بوصف ما تحتاجه بلغة طبيعية. لا حاجة لأي كود. يمكن لـ **وكيل الذكاء الاصطناعي** أيضًا إنشاء محتوى بالجملة وتوليد الصور وترجمة المدخلات والمزيد — راجع صفحة [وكيل الذكاء الاصطناعي](/ar/studio/ai-agent/) للتفاصيل الكاملة.

## وكيل الذكاء الاصطناعي — مساعد ذكي

يشمل Jambo Studio **وكيل ذكاء اصطناعي** قويًا يتخطى بكثير مجرد توليد المخطط. يمتلك الوكيل 8 أدوات حقيقية:

- **`explore_schema`** — تحليل مجموعاتك وحقولك الحالية
- **`create_collections`** — إنشاء مجموعات + حقول في خطوة واحدة
- **`create_entries`** — إنشاء ما يصل إلى 100 مدخلة بالجملة × لغات متعددة
- **`update_entries`** — تعديل المحتوى الموجود
- **`delete_entries`** — الحذف الناعم للمدخلات
- **`generate_images`** — توليد الصور بالذكاء الاصطناعي (DALL‑E / Gemini) + رفع تلقائي
- **`translate_entries`** — ترجمة المحتوى بين اللغات باستخدام الذكاء الاصطناعي

يكتشف الوكيل قدرات مزودك (نص، صور) ويتكيّف تلقائيًا. اقرأ الدليل الكامل: [توثيق وكيل الذكاء الاصطناعي](/ar/studio/ai-agent/).

## كيفية عمل الدردشة

هناك وضعان:

### الوضع السريع (`/schema`، `/data`، `/all`)

للطلبات الصغيرة، استخدم الأوامر المسبوقة بشرطة مائلة. يرد الذكاء الاصطناعي بمعاينة JSON وزر "تطبيق".

```
/schema إنشاء مدونة بمقالات وفئات ومؤلفين
/data توليد 5 مقالات مدونة احترافية بالعربية
/all بناء مخطط متجر إلكتروني كامل مع بيانات نموذجية
```

### وضع الوكيل

للطلبات المعقدة أو الجماعية أو متعددة اللغات، يتحوّل الذكاء الاصطناعي إلى **وضع التخطيط**. يعرض خطة تنفيذ مع قائمة الإجراءات. راجع الخطة وانقر على **تنفيذ** لتشغيل كل شيء تلقائيًا.

```
إنشاء موقع وكالة كامل بـ 6 مشاريع محفظة،
و8 شهادات عملاء ومدونة — كل ذلك بـ 4 لغات.
```

يُنشئ الوكيل المجموعات ويولّد المدخلات ويرفع الصور — دون مغادرة نافذة الدردشة.

## اتفاقيات التسمية

يطبّق Jambo قواعد تسمية صارمة تلقائيًا:

- أسماء المجموعات: **PascalCase جمع** (`BlogPosts`، `Products`)
- أسماء المفردات: **PascalCase مفرد** (`Hero`، `Config`)
- أسماء الحقول: **camelCase** (`publishedAt`، `featuredImage`)
- الـ Slugs: **snake_case** (`blog_posts`، `published_at`)
- **EndUsers** مجموعة نظام مدمجة — لا تُنشئ مجموعة "Users" جديدة أبدًا

## أنواع الحقول المدعومة

`text` · `longtext` · `richtext` · `slug` · `email` · `password` · `number` · `decimal` · `boolean` · `date` · `datetime` · `time` · `color` · `json` · `enumeration` · `media` · `relation`

## العمليات الجماعية ومتعددة اللغات

يمكن لوكيل الذكاء الاصطناعي التعامل مع عمليات جماعية تستغرق ساعات يدويًا:

- **إنشاء 100 مدخلة بـ 4 لغات** (400 استدعاء API) — بنقرة واحدة
- **ترجمة مجموعة كاملة** من الفرنسية إلى الإنجليزية والإسبانية والعربية
- **توليد صور ذكاء اصطناعي** لأغلفة المدونات والأقسام الرئيسية وصور المؤلفين
- **تعديل عشرات المدخلات** دفعة واحدة (تحديث الفئات، تصحيح الـ slugs، إلخ.)

## قدرات المزود

يكتشف الوكيل ما يمكن لمزود الذكاء الاصطناعي الخاص بك فعله. تعرض شارة القدرات الحالية:

🟢 **نص** (نشط) · 🟢 **صور** (نشط) · 🔴 **صوت** (غير متاح)

هيّئ المزودين في [الإدارة ← إعدادات التطبيق ← مزودو الذكاء الاصطناعي](/admin/app-settings). يستخدم الوكيل تلقائيًا المزود الأول المُفعَّل ويتحوّل إلى التالي عند الفشل.
