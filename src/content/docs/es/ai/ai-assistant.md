---
title: Assistant IA
description: Utiliser l'assistant IA pour écrire et améliorer le contenu dans l'éditeur Jambo.
---

L'assistant IA est disponible dans l'éditeur de contenu pour vous aider à écrire, réécrire et améliorer du texte directement dans le panneau d'administration.

## Accéder à l'assistant

Cliquez sur le bouton **✨ IA** dans la barre d'outils supérieure de tout formulaire d'entrada de contenu. Une barre latérale s'ouvre où vous pouvez taper des instructions.

## Ce que vous pouvez faire

### Générer du contenu

```
Écris une introduction de 200 mots sur l'emballage durable
```

### Réécrire du contenu existant

Sélectionnez du texte dans n'importe quel campo, puis demandez à l'assistant de le réécrire :

```
Rends ça plus court et plus professionnel
```

## Joindre des archivos

Cliquez sur le bouton **📎** à côté de la saisie pour joindre un archivo. Formats supportés :

- **Images** (PNG, JPG, WebP) — l'IA analyse l'image visuellement (nécessite GPT-4o ou Claude)
- **CSV / JSON** — l'IA lit les données et peut créer des entradas depuis celles-ci
- **Archivos texte** (TXT, MD, PDF) — l'IA lit le contenu et répond à des questions

## Requisitos previos

Un provider IA doit être configuré dans **Admin → Paramètres → Studio IA**.
