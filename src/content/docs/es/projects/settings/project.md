---
title: Paramètres du proyecto
description: Configurer les paramètres généraux du proyecto dans Jambo.
---

Accédez aux paramètres du proyecto via **Paramètres → Proyecto** dans la barre latérale.

## Général

| Parámetro | Descripción |
|-----------|-------------|
| **Nom** | Nom d'affichage du proyecto |
| **Slug** | Identifiant URL-safe (le changer casse les appels API existants) |
| **Description** | Description optionnelle |
| **Idioma par défaut** | La langue principale utilisée quand aucune idioma n'est spécifiée dans les appels API |
| **Statut** | Actif ou archivé |

## Zone dangereuse

| Action | Descripción |
|--------|-------------|
| **Archiver le proyecto** | Masque le proyecto du tableau de bord ; l'API continue de fonctionner |
| **Supprimer le proyecto** | Supprime définitivement le proyecto, toutes les coleccións, entradas et archivos médias |

:::caution
La suppression d'un proyecto est irréversible. Toutes les données sont eliminadoes définitivement.
:::
