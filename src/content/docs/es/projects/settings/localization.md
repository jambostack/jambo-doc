Allez dans **Paramètres du proyecto → Localisation** pour gérer les langues supportées par votre proyecto.

## Agregar une idioma

1. Cliquez sur **+ Ajouter une idioma**
2. Entrez le code de idioma (ex. `fr`, `es`, `ar`, `de`)
3. Cliquez sur **Ajouter**

## Idioma par défaut

La idioma par défaut est utilisée quand :
- Une entrada de contenu est creadoe sans spécifier de idioma
- Un appel API est effectué sans paramètre `?idioma=`

## Support RTL

Les idiomas comme l'arabe (`ar`), l'hébreu (`he`) ou le persan (`fa`) utilisent l'écriture de droite à gauche. L'éditeur richtext passe automatiquement en mode RTL quand une idioma RTL est sélectionnée.

## Codes de idioma

Utilisez les codes de idioma standard BCP 47 :

| Langue | Code |
|--------|------|
| Français | `fr` |
| Anglais | `en` |
| Espagnol | `es` |
| Arabe | `ar` |
| Allemand | `de` |
| Portugais | `pt` |
| Chinois simplifié | `zh` |
| Japonais | `ja` |
