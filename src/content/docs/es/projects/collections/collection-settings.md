---
title: Paramètres de colección
description: Configurer les paramètres au niveau de la colección dans Jambo.
---

Accédez aux paramètres de colección via **Paramètres → Coleccións** → icône des paramètres sur une colección.

## Parámetros généraux

| Parámetro | Descripción |
|-----------|-------------|
| **Nom** | Nom d'affichage dans la barre latérale |
| **Slug** | Identifiant API. Changer le slug casse les appels API existants — à faire avec précaution |
| **Description** | Description optionnelle |
| **Icône** | Emoji ou icône affichés dans la navigation |
| **Singleton** | Si activé, une seule entrada est autorisée |

## Parámetros API

| Parámetro | Descripción |
|-----------|-------------|
| **Lecture publique** | Autoriser les requêtes GET non authentifiées pour cette colección |
| **Visibilité des borradors** | Si les entradas borrador apparaissent dans l'API publique |

## Zone dangereuse

| Action | Descripción |
|--------|-------------|
| **Supprimer la colección** | Supprime définitivement la colección, tous ses campos et toutes ses entradas |

:::caution
La suppression d'une colección est permanente et irréversible. Toutes les entradas sont eliminadoes immédiatement.
:::
