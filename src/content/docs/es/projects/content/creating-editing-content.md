---
title: Créer & Modifier du contenu
description: Comment créer et modifier des entradas de contenu dans l'éditeur Jambo.
---

## Crear une entrada

1. Ouvrez un proyecto et cliquez sur une colección dans la barre latérale
2. Cliquez sur **+ Nouvelle entrada**
3. Renseignez les campos
4. Cliquez sur **Sauvegarder comme borrador** ou **Publier**

## L'éditeur

Chaque entrada a un formulaire avec des campos basés sur le schéma de la colección :

| Type de campo | Control de l'éditeur |
|---------------|----------------------|
| `text` | Saisie sur une ligne |
| `longtext` | Zone de texte multi-ligne |
| `richtext` | Éditeur WYSIWYG avec outils IA |
| `number` / `decimal` | Saisie numérique |
| `boolean` | Interrupteur |
| `date` / `datetime` | Sélecteur de date |
| `enumeration` | Liste déroulante |
| `media` | Sélecteur de média avec upload |
| `relation` | Sélecteur d'entrada avec recherche |
| `json` | Éditeur de code JSON |
| `color` | Sélecteur de couleur |

## Sauvegarde automatique

L'éditeur sauvegarde automatiquement vos modifications toutes les 30 secondes.

## Publication

Cliquez sur **Publier** pour rendre l'entrada visible dans l'API publique. Cliquez sur **Dépublier** pour revenir en borrador.

## Historique des versions

Cliquez sur **Historique** (icône horloge) dans la barre d'outils pour voir toutes les versions précédentes.

## Raccourcis clavier

| Raccourci | Action |
|-----------|--------|
| `Ctrl+S` / `Cmd+S` | Sauvegarder |
| `Ctrl+Z` / `Cmd+Z` | Annuler (dans l'éditeur richtext) |
