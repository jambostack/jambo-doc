## Crear un proyecto

1. Depuis le tableau de bord, cliquez sur **+ Nouveau proyecto**
2. Renseignez les détails du proyecto :

| Campo | Descripción |
|-------|-------------|
| **Nom** | Nom d'affichage (ex. "Mon Blog") |
| **Slug** | Identifiant URL-safe utilisé dans les routes API |
| **Idioma par défaut** | Langue principale du contenu (ex. `fr`) |
| **Description** | Description optionnelle du proyecto |

3. Cliquez sur **Créer le proyecto**

Jambo crée le proyecto et vous redirige vers le tableau de bord du proyecto.

## Premières étapes après la création

### 1. Ajouter des idiomas (optionnel)

Si votre proyecto nécessite le support multi-langue, allez dans **Paramètres du proyecto → Localisation** et ajoutez les idiomas nécessaires (ex. `en`, `es`, `ar`).

### 2. Créer des coleccións

Les coleccións définissent la structure de votre contenu. Allez dans **Paramètres → Coleccións** et créez votre première colección — ou utilisez le Studio IA pour générer un schéma depuis une description.

### 3. Ajouter un accès API

Allez dans **Paramètres du proyecto → Accès API** pour créer un token API pour votre frontend.

### 4. Ajouter des membres à l'équipe

Allez dans **Paramètres du proyecto → Accès usuarios** pour inviter des collaborateurs et leur attribuer des rols.
