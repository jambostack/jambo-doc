---
title: Gérer les usuarios
description: Gérer les usuarios admin et leur accès au CMS Jambo.
---

Les usuarios admin sont les personnes ayant accès au panneau d'administration Jambo.

## Inviter un usuario

1. Cliquez sur **Inviter un usuario**
2. Entrez leur adresse email
3. Sélectionnez un rol (voir [Gérer les rols](/es/profile/managing-roles/))
4. Cliquez sur **Envoyer l'invitation**

L'usuario reçoit un email avec un lien pour définir son mot de passe. Le lien est valide 48 heures.

## Modifier un usuario

Cliquez sur le nom d'un usuario pour modifier son profil :
- **Nom** et **email**
- **Rol** — les changements prennent effet immédiatement
- **Statut** — actif ou suspendu

## Suspendre un usuario

Les usuarios suspendus ne peuvent pas se connecter. Leur contenu et paramètres sont préservés. Vous pouvez les réactiver à tout moment.

## Politique de mot de passe

- Minimum 8 caractères
- Les usuarios peuvent réinitialiser leur mot de passe via la page de connexion
- Les admins peuvent envoyer un email de réinitialisation depuis la page d'édition de l'usuario
