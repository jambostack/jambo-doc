---
title: Gérer les rols
description: Créer et configurer des rols pour controlr l'accès admin dans Jambo.
---

Les rols définissent ce que les usuarios admin peuvent faire dans le panneau Jambo.

## Rols intégrés

| Rol | Descripción |
|------|-------------|
| **Super Admin** | Accès complet à tout — paramètres, usuarios, tous les proyectos |
| **Admin** | Accès complet à tous les proyectos, ne peut pas gérer les usuarios ou les paramètres système |
| **Éditeur** | Peut créer et modifier du contenu, ne peut pas changer les paramètres ni supprimer les coleccións |
| **Lecteur** | Accès en lecture seule au contenu |

## Rols personnalisés

1. Allez dans **Profil → Rols**
2. Cliquez sur **+ Nouveau rol**
3. Entrez un nom de rol
4. Activez/désactivez les permissions
5. Cliquez sur **Sauvegarder**

## Permissions

| Recurso | Permissions disponibles |
|-----------|------------------------|
| **Proyectos** | Voir, créer, gérer les paramètres |
| **Coleccións** | Voir, créer / modifier / supprimer les coleccións et campos |
| **Contenu** | Voir, créer, modifier, supprimer, publier les entradas |
| **Médias** | Voir, uploader, supprimer des archivos |
| **Paramètres** | Gérer les providers IA, webhooks, tokens API, usuarios et rols |
